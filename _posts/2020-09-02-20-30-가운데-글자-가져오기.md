<strong>문제 설명</strong>

{% highlight txt %}
단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요.
단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.
{% endhighlight %}

<strong>제한사항</strong>
{% highlight txt %}
s는 길이가 1 이상, 100이하인 스트링입니다.
{% endhighlight %}

| 입출력 | 예 |
| --- | --- |
| s	| return |
| abcde |	c |
| qwer | "we" |

{% highlight cpp %}
#include <string>
#include <vector>

using namespace std;

string solution(string s) {
    string answer = "";
        
    if (s.size() % 2)
        answer += s[s.size() / 2];
    else {
        answer += s[s.size() / 2 - 1];
        answer += s[s.size() / 2];
    } 
    return answer;
}
{% endhighlight %}
